# 使用方法

1. 克隆仓库到本体

   ```
   $ git clone
   ```

2. 安装依赖

   ```
   $ npm install
   ```

3. 本地运行

   执行下面命令，然后打开浏览器，地址栏输入`http://localhost:4321/`

   ```
   $ npm run start
   ```

# 文件目录介绍

1. 后端代码

   /bin   网站启动文件

   /database 数据库文件

   /models  后端应用文件

   /routes  路由文件

   /app.js  应用的入口文件

   /views  ejs模板文件

2. 前端代码

   /src  开发代码文件

   /public 发布代码文件
